 GEOS - Gerritsen Energy Operating System

GEOS is a fully local, modular Energy Operating System for Home Assistant.

The goal of GEOS is to optimize energy flows inside homes and businesses using adaptive control, AI-assisted optimization and a plugin-based architecture.

GEOS supports:

- Solar Inverters
- Batteries
- EV Chargers
- Dynamic Pricing
- Forecasting
- Smart Home Devices

Everything runs locally.

No cloud required.
